A number of courses are available at Wroclaw QA School. Please find details below:

Introduction to Quality Assurance

Target audience of this course are those, who want to join IT.

Topics:
Module 1. Introduction into IT.
Module 2. Software Development Life Cycle.
Module 3. Testing Principles and Types.
Module 4. Test Design Techniques.
Module 5. Requirements Analysis.
Module 6. QA Documentation.
Module 7. Bugs.
Module 8. Web Testing.
Module 9. Mobile Testing.
Module 10. Agile: Scrum Team Game.
Module 11. Jira - Practical Course.
Module 12. Introduction into Test Automation.

Bonuses:
1. CV creation rules.
2. Recommendations to top students.

More information.

Basics of Web Automation: Java and Selenium WebDriver

Target audience of this course are those, who work as manual QA engineer and want to develop themselves in Test Automation.

Topics:
Module 1. Introduction. Theory.
Module 2. Java basics.
Module 3. Selenium WebDriver.
Module 4. Locators.
Module 5. PageObject Pattern and project structure.
Module 6. Unit-testing frameworks (TestNG/JUnit).
Module 7. Build instruments (Maven/Ant/Gradle).
Module 8. Practice.
Module 9. Selenium tips and tricks + Practice.
Module 10. Exam.

Bonuses:
1. Java Database Connection Module.
2. HTTP and JSON (API testing with RestAssured framework).
3. Java Mail module.

More information.

Basics of Mobile Automation: Java and Appium

Target audience of this course are those, who work as manual QA engineer and want to develop themselves in Test Automation.

Topics:
Module 1. Introduction. Theory.
Module 2. Java basics.
Module 3. Appium framework.
Module 4. Environment setting.
Module 5. Searching for elements.
Module 6. Page Object Pattern, Page Element Pattern. Project structure (Part 1).
Module 7. Page Object Pattern, Page Element Pattern. Project structure (Part 2).
Module 8. WEB-application automation on mobile devices.
Module 9. Tips and tricks.
Module 10. Exam.

Bonuses:
1. Java Database Connection Module.
2. HTTP and JSON (API testing with RestAssured framework).
3. Java Mail module.
